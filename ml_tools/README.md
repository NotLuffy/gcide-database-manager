# Machine Learning Tools Folder

This folder contains machine learning-related scripts and data files.

## Files

### ML Scripts
- **ml_analyzer.py** - Machine learning analyzer for pattern detection
- **ml_dimension_extractor.py** - Extracts dimensions using ML models
- **ml_validation_analyzer.py** - Validates using ML predictions

### ML Data
- **ml_dimension_predictions.csv** - CSV file with ML dimension predictions
- **ml_dimension_models.pkl** - Trained ML models (stored in root folder)

## Usage

These tools are experimental and supplement the main parser. The primary parsing logic is in `improved_gcode_parser.py` in the root folder.
