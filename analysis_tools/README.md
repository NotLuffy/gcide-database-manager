# Analysis Tools Folder

This folder contains advanced analysis scripts and data files for pattern detection and dimension extraction.

## Files

### Analysis Scripts
- **ml_analyzer.py** - Pattern detection and anomaly analysis
- **ml_dimension_extractor.py** - Extracts dimensions using trained models
- **ml_validation_analyzer.py** - Validation using predictive models

### Data Files
- **ml_dimension_predictions.csv** - CSV file with dimension predictions
- **ml_dimension_models.pkl** - Trained analysis models (pickle file)

## Usage

These tools are experimental and supplement the main parser. The primary parsing logic is in `improved_gcode_parser.py` in the root folder.
